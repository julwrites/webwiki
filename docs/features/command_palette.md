# Features Documentation

## Command Palette

The Command Palette allows users to quickly navigate, execute commands, and search content using a keyboard-centric interface.

### Usage

-   **Open**: Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux).
-   **Close**: Press `Escape` or click outside the modal.
-   **Navigate**: Use `ArrowUp` and `ArrowDown` to select items.
-   **Execute**: Press `Enter` to run the selected command or navigate to the selected page.

### Features

-   **Navigation**: Quickly jump to the Home page.
-   **Theme**: Toggle between Light and Dark modes.
-   **Search**: Real-time search of wiki content.
